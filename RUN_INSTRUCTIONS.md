# Running the Expense Tracker Project

To run this Django project, follow these steps in your terminal:

1. **Navigate to the project directory** (if you aren't already there):
   ```bash
   cd /Users/adarshsingh/AIADS/Fullstack
   ```

2. **Activate the virtual environment**:
   ```bash
   source env/bin/activate
   ```

3. **Run the development server**:
   ```bash
   python manage.py runserver
   ```

4. **Access the application**:
   Open your browser and go to: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

### Admin Access
If you want to access the Django Admin panel, go to [http://127.0.0.1:8000/admin/](http://127.0.0.1:8000/admin/).

---
> [!TIP]
> If you encounter any "Command not found" errors, ensure you have Python installed and the virtual environment is correctly activated.
